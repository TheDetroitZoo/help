Editing a Press Release

Publishing a Press Release is very simple. All you need to do is:
  1. Enter the title of the Press Release
  2. Enter a subheading of the article, if one exists
  3. Enter the contents of the article into the main body
  4. Enter the City / State for the Press Release
  5. If you don't want the location to show, check the "Hide" box
  6. Hit the big "Publish" button and you're done!

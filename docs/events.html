Creating or Editing an Event post

The event post type handles the display of public Zoo Events like Wild Lights, Zoo Brew, or Zoo Boo. It has easy-to-use fields for each thing most events would need including
  1. Event name (post title)
  2. Basic description (post content)
  3. Purchase button, link
  4. Images:
    1. Featured image - top banner - 1600x400 (retina, displays at 800x200)
    2. Mobile image - top banner on phones - 800x400 (retina, displays at 400x200)
    3. Archive image - square for use on the Zoo Events landing page -600x600
  5. Details - repeater field with Title/Content fields - use this for addional details like ticket levels, parking info, etc.
  6. FAQ - repeater with Question/Answer fields
  7. Sponsors - sponsors can be added with name, logo, link, and can be organized into levels.

It's fun and easy. Give it a try!
